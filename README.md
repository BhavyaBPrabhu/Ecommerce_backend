**ECOMMERCE BACKEND APPLICATION**  

A scalable backend application for an e-commerce platform built using **Java 17** and **Spring Boot**. 
Implements secure REST APIs, role-based access control, and core e-commerce workflows such as product management,catalog management, cart handling and order management.

**Tech Stack**
- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security (JWT)
- Maven
- Swagger / OpenAPI
- PostgreSQL
  
**Features Implemented**
- Developed JWT-based authentication with stateless session management
- Implemented Role-Based Access Control (ADMIN / USER) using Spring Security
- Designed product and category management APIs with search, pagination and sorting
- Built Cart management APIs which can add or remove items, update quantities, and prepare cart for checkout
- Built order management APIs to convert a cart to order at checkout, calculate total amount, empty the cart after order placement and handle order cancellation
- Secured REST APIs with method-level and URL-level authorization
- Documented APIs using Swagger/OpenAPI
- Followed clean layered architecture with DTO and mapper patterns
